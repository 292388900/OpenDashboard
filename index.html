Context Mapping

Represents the relationship between an external context (e.g. a course or course section) and a set of OpenDashboard dashboards.

Functions

getId

Get the primary key

getKey

Get the consumer key

getDashboards

Get the array of dashboard objects associated with this context

getContext

Get the external context id (e.g. course id) associated with this mapping

getModified

Get last modifed timestamp

addDashboard

Add a new dashboard to this context mapping. This is a framework function - you probably don't need to call it yourself.

Parameters
  • dashboard : Object

Course

Represents an external context (e.g. an LMS course)

Functions

getId

Get the external context id (e.g. course id)

getTitle

Get the context title

getInstructors

Get an array of Member objects containing the instructors for the context

getLearners

Get an array of Member objects containing the learner for the context

getEvents

Get an array of Event objects for this context

getEventsMedian

Get an integer representing the median number of events for learners in the context. This is used to determine a relative activity level for each learner.

setEventsMedian (deprecated)

Set the integer representing the median number of events

fromLTI

Create a Course from the inbound LTI launch object

Parameters
  • ltiLaunch : Object

buildRoster

Construct the course roster from an array of Members

Parameters
  • members : Array[Member]

addEvent

Add a new event to the array of events for this context

Parameters
  • event : Event
